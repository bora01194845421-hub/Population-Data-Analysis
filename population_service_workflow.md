# 경기도 인구자료 정리 서비스 — 워크플로우 & 서브에이전트 설계

> **대상 지역:** 전국 / 경기도 / 수원시 / 용인시 / 성남시 / 고양시 / 화성시  
> **수원시 추가 기능:** 행정동별 인구 + 생애주기 분류 출력  
> **데이터 수집 방식:** API 없이 다운로드 파일 업로드 방식

---

## 1. 전체 워크플로우 개요

```
[사용자]
   │
   ▼
[Step 1] 파일 업로드
   │  ─ 통계청 KOSIS CSV/Excel 다운로드 후 업로드
   │
   ▼
[Step 2] Orchestrator Agent
   │  ─ 파일 분류 및 에이전트 라우팅
   │
   ├──► [Agent A] 전국/경기/시단위 인구 정리
   │
   ├──► [Agent B] 수원시 행정동 인구 정리
   │
   └──► [Agent C] 수원시 생애주기 분석
          │
          ▼
   [Step 3] 결과 통합 & 출력
          │  ─ 통합 리포트 생성 (Excel / MD / HTML)
          ▼
        [사용자]
```

## 3. 에이전트 상세 설계

---

### 🔵 Orchestrator Agent (조율 에이전트)

**역할:** 업로드된 파일을 분류하고 각 서브에이전트에 작업을 위임

**입력:** 사용자 업로드 파일 목록

**처리 단계:**

```
1. 파일명 및 헤더 파싱
   └─ 파일 종류 판별 (전국/경기/시단위/행정동/연령별)

2. 유효성 검사
   └─ 필수 컬럼 존재 확인 (지역, 연령, 인구수, 기준년월)
   └─ 결측 파일 사용자에게 안내

3. 에이전트 라우팅
   └─ 전국/경기/시 데이터 → Agent A
   └─ 수원 행정동 데이터 → Agent B
   └─ 수원 연령별 데이터 → Agent C

4. 결과 수집 및 통합 리포트 생성
```

**출력:** 에이전트별 작업 지시 + 통합 결과

---

### 🟢 Agent A — 전국/경기/시 인구 정리

**역할:** 전국 → 경기도 → 5개 시 수준의 인구 현황 정리

**입력:** 시군구 단위 연령별 인구 CSV

**처리 단계:**

```
Step A-1. 데이터 로드 & 정제
  - 기준년월 최신 데이터 필터링
  - 지역코드 기준으로 대상 지역 필터 (전국/경기/수원/용인/성남/고양/화성)
  - 결측값, 이상값 처리

Step A-2. 집계
  - 총인구 / 남성인구 / 여성인구 / 성비
  - 전년 대비 증감 (전년도 파일 존재 시)
  - 인구밀도 (면적 테이블 조인 필요 시 사용자 안내)

Step A-3. 비교 테이블 생성
  - 7개 지역 나란히 비교
  - 전국/경기 대비 각 시의 비율(%) 계산

Step A-4. 출력
  - 요약 테이블 (Markdown + Excel 시트)
```

**출력 예시:**

| 지역 | 총인구 | 남성 | 여성 | 성비 | 전년대비 |
|------|--------|------|------|------|----------|
| 전국 | 51,439,038 | 25,634,000 | 25,805,038 | 99.3 | -0.18% |
| 경기도 | 13,649,394 | 6,874,000 | 6,775,394 | 101.5 | +0.52% |
| 수원시 | 1,193,131 | 599,000 | 594,131 | 100.8 | -0.21% |
| 용인시 | 1,081,154 | 543,000 | 538,154 | 100.9 | +0.44% |
| 성남시 | 925,310 | 462,000 | 463,310 | 99.7 | -0.83% |
| 고양시 | 1,076,074 | 530,000 | 546,074 | 97.1 | -0.31% |
| 화성시 | 928,161 | 474,000 | 454,161 | 104.4 | +1.62% |

### 🟠 Agent B — 수원시 행정동별 인구 정리

**역할:** 수원시 4개 구, 44개 행정동 단위 인구 상세 분석

**입력:** 수원시 행정동별 연령별 인구 CSV

**처리 단계:**

```
Step B-1. 행정구역 계층 구성
  - 수원시 → 장안구/권선구/팔달구/영통구 → 각 행정동

Step B-2. 행정동별 집계
  - 총인구 / 남녀 / 세대수 / 세대당 인구
  - 구별 소계 및 시 전체 합계

Step B-3. 순위 분석
  - 인구 상위 5개 행정동
  - 인구 하위 5개 행정동
  - 구별 인구 비중(%)

Step B-4. 출력
  - 구별 요약 테이블
  - 행정동 전체 목록 테이블 (구 구분 포함)
```

**출력 예시:**

| 구 | 행정동 | 총인구 | 남 | 여 | 세대수 |
|----|--------|--------|-----|-----|--------|
| 장안구 | 파장동 | 28,451 | 14,102 | 14,349 | 11,832 |
| 장안구 | 정자1동 | 22,103 | 10,841 | 11,262 | 9,241 |
| … | … | … | … | … | … |
| 영통구 | 망포2동 | 41,882 | 20,934 | 20,948 | 15,621 |

### 🔴 Agent C — 수원시 생애주기 인구 분석

**역할:** 연령 구간을 생애주기로 재분류하여 수원시 인구 특성 분석

**입력:** 수원시 (+ 행정동별) 연령별 인구 CSV

#### 생애주기 분류 기준

| 생애주기 단계 | 연령 구간 | 설명 |
|---|---|---|
| 영유아 | 0 ~ 6세 | 어린이집·유치원 대상 |
| 아동 | 7 ~ 12세 | 초등학교 |
| 청소년 | 13 ~ 18세 | 중·고등학교 |
| 청년 | 19 ~ 34세 | 대학·취업·결혼 준비기 |
| 중장년 | 35 ~ 49세 | 경제활동 핵심층 |
| 장년 | 50 ~ 64세 | 은퇴 준비기 |
| 노년 전기 | 65 ~ 74세 | 활동적 노년 |
| 노년 후기 | 75세 이상 | 돌봄 수요층 |

**처리 단계:**

```
Step C-1. 연령 구간 집계
  - 1세 단위 데이터를 생애주기 구간으로 합산

Step C-2. 비율 계산
  - 각 생애주기 인구 / 총인구 × 100

Step C-3. 부양비 지표 계산
  - 유소년부양비 = (0~14세 / 15~64세) × 100
  - 노년부양비   = (65세이상 / 15~64세) × 100
  - 총부양비     = 유소년부양비 + 노년부양비
  - 고령화지수   = (65세이상 / 0~14세) × 100

Step C-4. 행정동별 생애주기 분포 (Agent B 결과와 결합)
  - 행정동별 주요 생애주기 비율 계산
  - 아동 밀집 행정동 / 고령 밀집 행정동 식별

Step C-5. 시계열 변화 (복수 연도 파일 제공 시)
  - 연도별 생애주기 인구 추이

Step C-6. 출력
  - 생애주기별 인구 요약 테이블
  - 부양비 지표 테이블
  - 행정동별 생애주기 히트맵 (Excel 조건부 서식)
```

**출력 예시 — 생애주기별 인구:**

| 생애주기 | 연령 | 인구 | 비율 | 전년대비 |
|----------|------|------|------|----------|
| 영유아 | 0~6세 | 52,341 | 4.4% | -2.1% |
| 아동 | 7~12세 | 61,204 | 5.1% | -1.8% |
| 청소년 | 13~18세 | 58,992 | 4.9% | -0.5% |
| 청년 | 19~34세 | 241,803 | 20.3% | -1.2% |
| 중장년 | 35~49세 | 302,115 | 25.3% | -0.4% |
| 장년 | 50~64세 | 268,442 | 22.5% | +1.3% |
| 노년 전기 | 65~74세 | 124,908 | 10.5% | +3.1% |
| 노년 후기 | 75세~ | 83,326 | 7.0% | +4.2% |
| **합계** | | **1,193,131** | **100%** | |

**출력 예시 — 부양비 지표:**

| 지표 | 수원시 | 경기도 | 전국 |
|------|--------|--------|------|
| 유소년부양비 | 19.8 | 20.4 | 18.6 |
| 노년부양비 | 24.3 | 22.1 | 27.0 |
| 총부양비 | 44.1 | 42.5 | 45.6 |
| 고령화지수 | 122.7 | 108.3 | 145.2 |

## 6. 확장 고려사항

### 6-1. 시계열 분석 추가
- 복수 연도 파일 업로드 시 Agent C에서 자동으로 연도별 추이 생성
- 인구 감소/증가 변곡점 자동 표시

### 6-2. 다른 시 행정동 데이터 확장
- 현재 수원시만 행정동 지원 → 용인/성남/고양/화성도 동일 구조로 Agent B 재사용 가능

### 6-3. 정책 수요 연계
- 생애주기 결과를 공공시설 수요(어린이집, 경로당, 청년센터 등)와 연계하는 분석 레이어 추가 가능

### 6-4. 자동화 전환
- 추후 API 방식으로 전환 시: 행안부 공공데이터포털 `jumin.mois.go.kr` REST API 연동으로 교체
